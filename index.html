Passos para Hospedar um Projeto no Github Pages

  1. Crie um repositorio no Github com o mesmo nome do seu projeto.
  2. No terminal, inicialize o respeito local com git init.
  3. Altere o nome da branch principal para main
  4. Adicione os arquivos com o git add . e faça o commit com git commit -m "first commit"
  5. Conecte ao repositório remoto com:
    git remote add origin https://github.com/SEU_USUARIO/NOME_DO_REPOSITORIO.git
  6. Envie os arquivos com git push -u origin main
  7. No GitHub, vá em Setting > Pages
  8. Em "Branch", selecione main e a pasta
